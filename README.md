# Sorting-Visualizer

This project is a Web Visualization tool for sorting algorithms.

- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort

[Link](https://maverick-shooter.github.io/Sorting-Visualizer/)
